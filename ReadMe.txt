=============================================================
RawRenamer ver.0.1
=============================================================

機能
Canon社製デジタル一眼レフカメラのRAWファイル(CRW,CR2)を撮影日時でリネームします。

特徴 
標準では連番形式等になってしまうRAWファイルを、
ファイル中の撮影日時情報を取得してリネームします。
サブフォルダのファイルもまとめて処理。 
日付形式は日時を入れるタイプ[yymmdd_hhmmss]と時刻のみのタイプ[hhmmss]を用意しました。
フリーソフトです。 

動作環境 
Windows XP/2000

インストール
解凍したファイルをダブルクリックすれば起動します（インストーラ等はありません） 

アンインストール
インストール時に出来たファイルと
設定ファイル(RawRenamer.ini)を削除してください（レジストリ等は使用していません）。 

使い方 
１．フォルダを指定してください。 
２．日付形式を指定してください。
３．開始ボタンを押してください。
４．リネーム中のファイルが表示されるのでしばしお待ちください。 
５．中断ボタンを押すことで途中で処理を終了することも出来ます
　　（ファイルのリネーム中の場合は、中断を押した時にリネームしているファイルまでリネームされます）。
６．全てリネームされれば終了です。 

その他 
RAWファイルの情報等は非公開のため、作者が独自に解析を行った結果を元にしています。
そのため、正常な結果にならない場合もございますのであらかじめご了承ください。
EOS 10D(CRW), EOS 20D(CR2)のRAWファイルで確認を行っております。
